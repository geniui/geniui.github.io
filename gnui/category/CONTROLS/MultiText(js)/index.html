생성 옵션

<div class="default-multitext">
</div>
<div class="new-multitext">
</div>

Gn.create('multitext', {
  target: '.default-multitext'
})

Gn.create('multitext', {
  target: '.new-multitext',
  lang: ['en','ko','cn'],
  value: {
    en: 'hello',
    ko: '안녕하세요',
    cn: 'HELLO'
  },
  maxlength: 300
})
name type default description
target string multitext 생성 타겟
lang Array ['en','ko'] 사용할 언어 배열
value Object 언어별 초기 값
size string 'normal' 사이즈
width string 100% 넓이
rows number 2 textarea 라인 수
maxlength number 524288 최대 입력 글자길이
onChange(value) function 값 변경 시 발생 이벤트
disabled boolean false 비활성화 여부

메소드

name return description
getValue() String 값을 반환한다.
disable() void Disabled 처리한다.
enable() void Enabled 처리한다.
