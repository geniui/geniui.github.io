기본 사용

  • 사용 태그 : input[text], input[password], input[search], input[email], input[tel]
  <input type="text" class="gn-input" placeholder="text input">
  <input type="password" class="gn-input" placeholder="password input">
  <input type="search" class="gn-input" placeholder="search input">
  <input type="tel" class="gn-input" placeholder="tel input">

<input type="text" class="gn-input" placeholder="primary text : border 1px">
<input type="text" class="gn-input is-border-width2" placeholder="primary text : border 2px"> 
<input type="text" class="gn-input is-border-width3" placeholder="primary text : border 3px">
