name type default description
target string 컴포넌트 생성대상 선택자(selector)
width string 컴포넌트 넓이
data Array[{text, value}] 출력할 태그 목록
value 값에 따라 1~9 단계로 text를 표시한다.
onClick function(data) 태그 클릭 시 수행될 이벤트
