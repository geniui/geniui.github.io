생성 옵션

  <div class="gn-container page-message"></div>
  <button class="gn-button new-message">Message 예제보기</button>

  Gn.create('message', {
    target: '.page-message', 
    textSets: { message: '메세지가 생성됩니다!<br>잠시만 기다려주세요.' }
  });
name type default description
target string 메시지를 표시할 대상의 선택자(selector)
textSets message string 메시지에 표기할 문구
color string 'success','warning','danger','info','cancel','dark','link','trans','secondary','mono'
size string 'small','normal','medium','large'
icon string font awesome solid icon name
animation string slideup 'slideup','fadeout'
duration number 300 메시지 닫기 이벤트 애니메이션 시간
hasClose boolean true 닫기버튼 출력 여부
onClose function 닫기버튼 클릭 시 수행될 이벤트

메소드

name return description
close() void 메세지를 닫는다.
